# Web Dev Learning 🚀

A personal repository for learning and practicing web development.

## Topics Covered

- HTML structure and semantics
- CSS styling and layouts
- JavaScript fundamentals
- Responsive design

## Structure

```
web-dev-learning/
├── index.html        # Main HTML starter page
├── css/
│   └── style.css     # Stylesheet
├── js/
│   └── app.js        # JavaScript
└── README.md
```

## Getting Started

Clone the repo and open `index.html` in your browser.

```bash
git clone https://github.com/kingldtech/web-dev-learning.git
```
